# BizSync

An AI Driven application that is going to help different local business to manage their clients and respond them instantly on whatsapp. This is a SaaS project.

## Getting Started

The application is starting with onboarding with welcoming users to create their business, and configure their business working hours to get in touch with their potential clients. The chat message that will be sent to the client might have different types such as who sent the message? **AI?** **Human?** **Customer?**

## Frontend Mobile Application

At the frontend, flutter will be used for smooth and fast development. State management, API Handling and Background Application running will be handled inside flutter.

## Backend Application

At the backend, NodeJS along with MySQL will be used for smooth communication with Meta Business API and the AI Model to respond to customers immediately,
